4 Function types

Any two types \(X\) and \(Y\) give us a family of types \begin{equation*}(Y_{x})_{x \mathrel{:}X}\end{equation*} on \(X\), where \begin{equation*}Y_{x} \equivY\end{equation*} for all \(x \mathrel{:}X\). We call this the constant family on \(X\) with value \(Y\).

A selection for this family is called a function from \(X\) to \(Y\). It is an assignment \(f\) that attaches an element \begin{equation*}f(x) \mathrel{:}Y\end{equation*} to every element \(x \mathrel{:}X\).

The Pi type of the family is denoted by \begin{equation*}X \toY.\end{equation*} It is called the function type of \(X\) and \(Y\). It is an element of the fixed universe \(\mathscr{U}\). Its elements are exactly the functions from \(X\) to \(Y\).

For any function \begin{equation*}f \mathrel{:}X \toY,\end{equation*} the type \(X\) is called the domain of \(f\) and is denoted by \(\operatorname{Dom}(f)\), while \(Y\) is called the codomain of \(f\) and is denoted by \(\operatorname{Cod}(f)\).

Given an element \begin{equation*}y_{x} \mathrel{:}Y\end{equation*} for every element \(x \mathrel{:}X\), we have the abstraction \begin{equation*}\lambda{}(x \mathrel{:}X).\, y_{x} \mathrel{:}X \toY,\end{equation*} which is a function with the property that \begin{equation*}(\lambda{}(x \mathrel{:}X).\, y_{x}) (a) \equivy_{a}\end{equation*} for all \(a \mathrel{:}X\).

Conversely, every function \begin{equation*}f \mathrel{:}X \toY\end{equation*} is the same as its abstraction: \begin{equation*}f \equiv\lambda{}(x \mathrel{:}X).\, f(x).\end{equation*} This fact is called \(\eta\)-conversion for functions.
