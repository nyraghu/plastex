11 The unit type

We are given a type \(\mathbf{1}\) called the unit type, and an element \begin{equation*}\star\mathrel{:}\mathbf{1}\end{equation*} called the canonical element of \(\mathbf{1}\).

For every family \begin{equation*}F \mathrel{:}\mathbf{1}\to\mathscr{U}\end{equation*} on \(\mathbf{1}\), and for every element \begin{equation*}a \mathrel{:}F(\star),\end{equation*} we are given a selection \(\phi_{F,a}\) for \(F\) with the property that \begin{equation*}\phi_{F,a}(\star) \equiva.\end{equation*} We call \(\phi_{F,a}\) the selection for \(F\) induced by the element \(a\).

Taking \(F\) to be a constant family, for every type \begin{equation*}X \mathrel{:}\mathscr{U},\end{equation*} and every element \begin{equation*}a \mathrel{:}X,\end{equation*} we get a function \begin{equation*}\phi_{a} \mathrel{:}\mathbf{1}\toX\end{equation*} such that \begin{equation*}\phi_{a}(\star) \equiva.\end{equation*} It is called the function from \(\mathbf{1}\) to \(X\) induced by \(a\).

We also have a function \begin{equation*}\psi_{X} \mathrel{:}X \to\mathbf{1}\end{equation*} defined by \begin{equation*}\psi_{X}(x) \equiv\star\end{equation*} for all \(x \mathrel{:}X\). It is called the canonical function from \(X\) to \(\mathbf{1}\).
