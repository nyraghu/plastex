2 Universes

Type theory begins by giving an infinite sequence \begin{equation*}\mathscr{U}_{0}, \mathscr{U}_{1}, \dotsc\end{equation*} of objects called universes. We will call the index \(i\) the grade of the universe \(\mathscr{U}_{i}\).

We are given that the sequence of universes has the following two properties:

  1. (Hierarchy) The object \(\mathscr{U}_{i}\) is an element of \(\mathscr{U}_{i+1}\) for every \(i\). In the notation introduced above, \begin{equation*}\mathscr{U}_{i} \mathrel{:}\mathscr{U}_{i+1}.\end{equation*}

  2. (Cumulativity) Every element of \(\mathscr{U}_{i}\) is an element of \(\mathscr{U}_{i+1}\) also. In symbols, if \(X \mathrel{:}\mathscr{U}_{i}\), then \(X \mathrel{:}\mathscr{U}_{i+1}\).

Let us now fix one of the given universes \(\mathscr{U}_{i}\), and denote it by \(\mathscr{U}\). An element of this fixed universe \(\mathscr{U}\) is called a type.

A family of types on a type \begin{equation*}X \mathrel{:}\mathscr{U}\end{equation*} is an assignment that attaches a type \begin{equation*}Y_{x} \mathrel{:}\mathscr{U}\end{equation*} to every element \(x \mathrel{:}X\). We denote this family by \begin{equation*}(Y_{x})_{x \mathrel{:}X}.\end{equation*} The type \(X\) is called the base of the family. If it is obvious from the context, we may simply write \begin{equation*}(Y_{x})\end{equation*} for the family.
