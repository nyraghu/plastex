6 Composition of functions

The composite of two functions \begin{equation*}f \mathrel{:}X \toY, \quad{}g \mathrel{:}X \toZ\end{equation*} is the function \begin{equation*}g \circf \mathrel{:}X \toZ\end{equation*} defined by \begin{equation*}g \circf \equiv\lambda{}(x \mathrel{:}X).\, g(f(x)).\end{equation*} When we vary \(f\) and \(g\), we get an element \begin{equation*}\phi\mathrel{:}(X \toY) \to(Y \toZ) \toX \toZ\end{equation*} given by \begin{equation*}\phi\equiv\lambda{}(f \mathrel{:}X \toY) (g \mathrel{:}Y \toZ).\, g \circf.\end{equation*}

We are using the convention that the scope of the lambda operator, as with the Pi operator, is the entire right of the expression unless stopped by parentheses. So \begin{equation*}\lambda{}(f \mathrel{:}X \toY) (g \mathrel{:}Y \toZ).\, g \circf\end{equation*} means \begin{equation*}\lambda{}(f \mathrel{:}X \toY) (g \mathrel{:}Y \toZ).\, (g \circf),\end{equation*} not \begin{equation*}(\lambda{}(f \mathrel{:}X \toY) (g \mathrel{:}Y \toZ).\, g) \circf.\end{equation*}

The definition of composition implies that for any three functions \begin{equation*}f \mathrel{:}X \toY, \quad{}g \mathrel{:}X \toZ, \quad{}h \mathrel{:}Z \toW,\end{equation*} we have \begin{equation*}((h \circg) \circf) (x) \equiv(h \circ(g \circf)) (x) \equivh(g(f(x)))\end{equation*} for all \(x \mathrel{:}X\). So by \(\eta\)-conversion we get the associativity law \begin{equation*}(h \circg) \circf \equivh \circ(g \circf).\end{equation*}

The same kind of verification gives the identity law \begin{equation*}f \circ\operatorname{id}_{X} \equiv\operatorname{id}_{Y} \circf \equivf\end{equation*} for every function \(f \mathrel{:}X \toY\).
