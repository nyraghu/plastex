8 Sigma types

A pair for a family \begin{equation*}F \mathrel{:}X \to\mathscr{U}\end{equation*} is an object \(t\) that consists of two other objects: an element \(x \mathrel{:}X\) called the first component of \(t\), and an element \(y \mathrel{:}F(x)\) called the second component of \(t\). Such a pair \(t\) is denoted by \begin{equation*}(x, y).\end{equation*}

We are given a type \begin{equation*}\Sigma{}(x \mathrel{:}X).\, F(x),\end{equation*} whose elements are exactly the pairs for the family \(F\). It is called the Sigma type of the family. By the convention that a type means an element of the fixed universe, the Sigma type is an element of \(\mathscr{U}\). We may simply write \begin{equation*}\Sigma{}x.\, F(x)\end{equation*} for the Sigma type if the base type \(X\) is obvious.

We have a function \begin{equation*}\operatorname{pr}_{1}\mathrel{:}\Sigma{}(x \mathrel{:}X).\, F(x) \toX\end{equation*} defined by \begin{equation*}\operatorname{pr}_{1}(x, y) \equivy\end{equation*} for all \(x \mathrel{:}X\) and \(y \mathrel{:}F(x)\). This function is called the first projection.

We also have a selection \begin{equation*}\operatorname{pr}_{2}\mathrel{:}\Pi{}(t \mathrel{:}\Sigma{}(x \mathrel{:}X).\, F(x)).\,F(\operatorname{pr}_{1}(t))\end{equation*} called the second projection. It is defined by \begin{equation*}\operatorname{pr}_{2}(x, y) \equivy\end{equation*} for all \(x \mathrel{:}X\) and \(y \mathrel{:}F(x)\).

In the other direction, for every \(a \mathrel{:}X\) there is a function \begin{equation*}\sigma_{a} \mathrel{:}F(a) \to\Sigma{}(x \mathrel{:}X).\, F(x)\end{equation*} defined by \begin{equation*}\sigma_{a}(y) \equiv(a, y)\end{equation*} for \(y \mathrel{:}F(a)\). It is called the canonical function from \(F(a)\) to the Sigma type.

The definitions of the two projections imply that \begin{equation*}(x,y) \equiv( \operatorname{pr}_{1}(x, y), \operatorname{pr}_{2}(x, y) )\end{equation*} for every pair \((x, y)\) for the family \(F\). In other words, \begin{equation*}t \equiv(\operatorname{pr}_{1}(t), \operatorname{pr}_{2}(t))\end{equation*} for every element \(t\) of \(\Sigma{}(x \mathrel{:}X).\, F(x)\). This relation is called \(\eta\)-conversion for Sigma types.

As in the case of Pi types, we use a shortened notation for iterated Sigma types. If \begin{equation*}F \mathrel{:}X \to\mathscr{U}\end{equation*} is a family of types, and if for every \(x \mathrel{:}X\), we are given a family \begin{equation*}G_{x} \mathrel{:}F(x) \to\mathscr{U},\end{equation*} we will denote \begin{equation*}\Sigma{}(x \mathrel{:}X).\, (\Sigma{}(y \mathrel{:}F(x)).\, G_{x}(y))\end{equation*} by \begin{equation*}\Sigma{}(x \mathrel{:}X) (y \mathrel{:}F(x)).\, G_{x}(y).\end{equation*} We use a similar notation for iterated Sigma types with more than two indices.
