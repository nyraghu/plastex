10 The empty type

We are given a type \(\mathbf{0}\) called the empty type. We are also given for every family \begin{equation*}F \mathrel{:}\mathbf{0}\to\mathscr{U}\end{equation*} on \(\mathbf{0}\), a selection \(\phi_{F}\) for \(F\), called the canonical selection for \(F\).

Now if \(X\) is any type, we can take \(F\) to be the constant family with domain \(\mathbf{0}\) and value \(X\). We then get a function \begin{equation*}\phi_{X} \mathrel{:}\mathbf{0}\toX,\end{equation*} which is called the canonical function from \(\mathbf{0}\) to \(X\).
