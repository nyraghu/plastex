14 Natural numbers

We are given a type \(\mathbf{N}\), an element \begin{equation*}\operatorname{0}\mathrel{:}\mathbf{N},\end{equation*} and a function \begin{equation*}\operatorname{S}\mathrel{:}\mathbf{N}\to\mathbf{N}.\end{equation*} The elements of \(\mathbf{N}\) are called the natural numbers. The natural number \(\operatorname{0}\) is called zero. For any \(n \mathrel{:}\mathbf{N}\), the natural number \(\operatorname{S}(n)\) is called the successor of \(n\).

For every family \begin{equation*}F \mathrel{:}\mathbf{N}\to\mathscr{U},\end{equation*} every element \begin{equation*}a \mathrel{:}F(\operatorname{0}),\end{equation*} and every element \begin{equation*}f \mathrel{:}\Pi{}(n \mathrel{:}\mathbf{N}).\, F(n) \toF(\operatorname{S}(n)),\end{equation*} we are given a selection \(\phi_{F,a,f}\) for \(F\) such that \begin{equation*}\phi_{F,a,f} (\operatorname{0}) \equiva,\end{equation*} and \begin{equation*}\phi_{F,a,f} (\operatorname{S}(n)) \equivf(n)(\phi_{F,a,f}(n))\end{equation*} for all \(n \mathrel{:}\mathbf{N}\). We call \(\phi_{F,a,f}\) the selection for \(F\) induced by \(a\) and \(f\).

When we specialise this dispensation to a constant family, for every type \begin{equation*}X \mathrel{:}\mathscr{U},\end{equation*} every element \begin{equation*}a \mathrel{:}X,\end{equation*} and every function \begin{equation*}f \mathrel{:}\mathbf{N}\toX \toX,\end{equation*} we get a function \begin{equation*}\phi_{a,f} \mathrel{:}\mathbf{N}\toX\end{equation*} such that \begin{equation*}\phi_{a,f} (\operatorname{0}) \equiva,\end{equation*} and \begin{equation*}\phi_{a,f} (\operatorname{S}(n)) \equivf(n)(\phi_{a,f}(n))\end{equation*} for all \(n \mathrel{:}\mathbf{N}\). It is called the function from \(\mathbf{N}\) to \(X\) induced by \(a\) and \(f\).
