12 The boolean type

We are given a type \(\mathbf{B}\) called the boolean type, and two elements \begin{equation*}\mathbf{t}\mathrel{:}\mathbf{B}, \quad{}\mathbf{f}\mathrel{:}\mathbf{B},\end{equation*} which are not the same. These two elements are called the canonical elements of \(\mathbf{B}\).

For every family \begin{equation*}F \mathrel{:}\mathbf{B}\to\mathscr{U},\end{equation*} and elements \begin{equation*}a \mathrel{:}F(\mathbf{t}), \quad{}b \mathrel{:}F(\mathbf{f}),\end{equation*} we are given a selection \(\phi_{F,a,b}\) for \(F\) with the property that \begin{equation*}\phi_{F,a,b}(\mathbf{t}) \equiva, \quad{}\phi_{F,a,b}(\mathbf{f}) \equivb.\end{equation*} It is called the selection for \(F\) induced by \(a\) and \(b\).

If we assume that \(F\) is a constant type, for every type \begin{equation*}X \mathrel{:}\mathscr{U},\end{equation*} and elements \begin{equation*}a \mathrel{:}X, \quad{}b \mathrel{:}X,\end{equation*} we get a function \begin{equation*}\phi_{a,b} \mathrel{:}\mathbf{B}\toX\end{equation*} such that \begin{equation*}\phi_{a,b}(\mathbf{t}) \equiva, \quad{}\phi_{a,b}(\mathbf{f}) \equivb.\end{equation*} We again call \(\phi_{a,b}\) the function from \(\mathbf{B}\) to \(X\) induced by \(a\) and \(b\).
